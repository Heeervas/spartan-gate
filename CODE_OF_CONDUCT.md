# Code Of Conduct

Spartan Gate follows the Contributor Covenant Code of Conduct, version 2.1.

## Our Pledge

We pledge to make participation in this project a harassment-free experience
for everyone, regardless of age, body size, visible or invisible disability,
ethnicity, sex characteristics, gender identity and expression, level of
experience, education, socio-economic status, nationality, personal appearance,
race, caste, color, religion, or sexual identity and orientation.

## Our Standards

Expected behavior includes:

- Using welcoming and inclusive language.
- Respecting differing viewpoints and experiences.
- Accepting constructive feedback.
- Focusing on what is best for the community and the project.

Unacceptable behavior includes harassment, insults, threats, sustained
disruption, publishing private information without permission, or sexualized
language or imagery in project spaces.

## Enforcement

Report conduct concerns through the repository owner contact path. Maintainers
may remove comments, issues, pull requests, or other contributions that do not
align with this code.

This code is adapted from the Contributor Covenant, version 2.1:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct.html>.
